I'll provide you with PDF document text and a folder path. Your task is to classify the document into one of our legal document categories and return a properly formatted JSON response.

# Document Categories

Please classify each document into one of these categories:

- CERL: Certificados de Existencia y Representación Legal
  These are certificates that prove a company's legal existence and representation.

- CECRL: Copia de cédulas de ciudadanía del Representante Legal
  These are copies of the legal representative's citizenship ID cards.

- RUT: Registro Único Tributario
  These are tax registration documents.

- RUB: Registro Único de Beneficiarios
  These are documents registering the beneficiaries of a company or entity.

- ACC: Composiciones Accionarias
  These documents detail the shareholder composition of a company.

# Document Path Structure

The folder path follows this pattern: {CATEGORY}/{DOCUMENT_NUMBER}/{FILENAME}

For example:
- "CECRL/5120000874/cedula_representante.pdf"
- "RUT/800003781/941_2020-02-29.pdf"
- "ACC/900266513/53_CA_2020-02-29.pdf"

# Important Path Instructions

When extracting the document_number:
- Extract the parent folder name (middle part of the path)
- In "CECRL/5120000874/cedula_representante.pdf", use "5120000874" as the document_number
- In "RUT/800003781/941_2020-02-29.pdf", use "800003781" as the document_number

When formatting the output path:
- Follow the pattern: "file_examples/{category}/{document_number}"
- Example: "file_examples/CECRL/5120000874"
- Exclude the filename from the path


# Special Cases

For empty documents or those with only whitespace, assign the "BLANK" category.

For documents containing only a hyperlink with no other meaningful text, assign the "LINK_ONLY" category.

# Expected Output

Provide a single JSON object with this structure:

{
  "document_number": "[subfolder name from the path]",
  "document_type": "[person or company]",
  "category": "[one of CERL|CECRL|RUT|RUB|ACC|BLANK|LINK_ONLY]",
  "text": "[all extracted text]",
  "path": "[document path from 'Folder path:']"
}

# Important Guidelines

1. Assign "person" as the document_type for CECRL documents and "company" for all other categories.
2. Use the exact string after 'Folder path:' for the path field.
3. Output only the JSON object without additional text or formatting.
4. Use the exact category spelling shown above, including underscores where needed.
